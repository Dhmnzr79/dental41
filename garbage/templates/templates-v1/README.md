# Старые шаблоны v1 (Legacy)

Эта папка содержит старые версии шаблонов страниц.

## Структура

- `page-implantatsiya.php` - Имплантация
- `page-istoriya-moei-ulybki.php` - История моей улыбки
- `page-contacts.php` - Контакты
- `page-blog.php` - Блог
- `page-sovety-kitay.php` - Советы для Китая
- `page-clinic-info.php` - Информация о клинике
- `page-thank-you.php` - Страница благодарности

## Как это работает

WordPress ищет шаблоны по имени файла в корне темы. 

В корне созданы **wrapper-шаблоны** с теми же именами, которые подключают старые шаблоны из этой папки.

Например:
- `/page-implantatsiya.php` (в корне) → подключает `/templates-v1/page-implantatsiya.php`

## Использование

Эти файлы используются для:
- Старых страниц, которые еще не мигрированы на v2
- Справки при разработке новых версий

## Миграция

Когда страница мигрируется на v2:
1. Создаем новую версию в корне (с v2 версткой)
2. Старый файл остается здесь для справки
3. Wrapper-шаблон в корне обновляется или удаляется
