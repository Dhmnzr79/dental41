# Старые шаблоны (Legacy)

Эта папка содержит старые версии шаблонов для справки.

**ВАЖНО:** WordPress не ищет шаблоны в подпапках автоматически. 
Эти файлы сохранены только для справки при разработке v2 версии.

## Структура

- `front-page.php` - старая версия главной страницы (для справки)

## Использование

Используйте эти файлы как справочник:
- Для понимания структуры контента
- Для копирования текстов (без изменения)
- Для понимания логики блоков

**НЕ копируйте:**
- Старые классы
- Старые стили
- Старую структуру сетки
