# Стили v2

Новая модульная структура CSS для версии 2 сайта.

## Структура файлов

```
v2/
├── base.css          # Переменные, reset, типографика (только для .v2-site)
├── layout.css        # Контейнер, сетка 12 колонок (только для .v2-site)
├── components.css    # БЭМ-компоненты (только для .v2-site)
└── utilities.css     # Утилиты (только для .v2-site)
```

## Правила

- ✅ Все стили обернуты в `.v2-site`
- ✅ БЭМ-именование с префиксом `v2-`
- ✅ Нет `!important`
- ✅ Контекст ≤ 1 уровень
- ✅ Медиа-запросы в конце файлов
- ✅ Брейкпоинты: 768px / 1280px / 1400px

## Подключение

Стили автоматически подключаются для `front-page.php` через `functions.php`.

## Компоненты

- `v2-hero` - Hero блок главной страницы
