# Список всех классов кнопок в схеме V2

## Основные классы кнопок

### 1. Базовый класс кнопки
- `.v2-btn` - основной класс кнопки (базовые стили)

### 2. Модификаторы основной кнопки
- `.v2-btn--primary` - основная кнопка (с градиентным фоном)
- `.v2-btn--secondary` - вторичная кнопка (с обводкой)
- `.v2-btn--whatsapp` - кнопка WhatsApp (зеленый фон)

### 3. Кнопки в формах
- `.v2-form-btn` - кнопка в формах (полная ширина, градиент)
- `.wpcf7-submit` - кнопка отправки Contact Form 7 (в попапах и форме консультации)
- `input[type='submit']` - стандартная кнопка отправки формы

## Специальные кнопки компонентов

### Хедер
- `.v2-header__icon-btn` - иконка-кнопка в хедере

### Слайдеры
- `.slider-arrow` - стрелка навигации слайдера (левая/правая)
- `.v2-slider-nav` - контейнер для стрелок слайдера
- `.v2-reviews__nav-btn` - навигационная кнопка слайдера отзывов

### Пагинация (точки)
- `.v2-works__dot` - точка пагинации блока "Наши работы"
- `.v2-reviews__dot` - точка пагинации блока "Отзывы"
- `.v2-doctors__dot` - точка пагинации блока "Врачи"
- `.v2-implants__dot` - точка пагинации блока "Импланты"

### Видео кнопки
- `.v2-reviews__video-btn` - кнопка воспроизведения видео в отзывах
- `.v2-doctors__video-btn` - кнопка воспроизведения видео в карточке врача

### Кнопки в карточках
- `.v2-works__cta-button` - CTA кнопка в блоке "Наши работы"
- `.v2-services__card-body .v2-btn` - кнопка в карточке услуги
- `.v2-implants__card-button` - кнопка в карточке импланта
- `.v2-doctor-selection__button` - кнопка в блоке подбора врача
- `.v2-china-risks__card--featured .v2-btn` - кнопка в featured карточке рисков

### Мобильные кнопки
- `.v2-hero-mobile__actions .v2-btn` - кнопка в мобильном hero блоке

## Структура файлов

### ui.css
- `.v2-btn` - базовые стили
- `.v2-btn--primary` - основная кнопка
- `.v2-btn--secondary` - вторичная кнопка
- `.v2-btn--whatsapp` - WhatsApp кнопка
- `.v2-form-btn` - кнопка формы
- `.v2-header__icon-btn` - иконка в хедере
- `.slider-arrow` - стрелки слайдера

### components.css
- Все компонентные кнопки (видео, CTA, карточки и т.д.)
- Кнопки пагинации (dots)
- Навигационные кнопки

## Использование

### Базовая кнопка
```html
<a href="#" class="v2-btn v2-btn--primary">Текст кнопки</a>
<button class="v2-btn v2-btn--primary">Текст кнопки</button>
```

### Вторичная кнопка
```html
<a href="#" class="v2-btn v2-btn--secondary">Текст кнопки</a>
```

### Кнопка в форме
```html
<button type="submit" class="v2-form-btn">Отправить</button>
```
